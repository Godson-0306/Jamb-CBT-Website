Exam instructions

Before you start

  • You will answer 200 objective questions in total.
  • Each selected subject contains 50 questions.
  • The timer starts immediately after you begin the test.
  • Use the question palette to jump quickly between questions.

Mouse controls

  • Click any option A to D to select an answer.
  • Click a numbered question in the palette to jump to it.
  • Use the Previous and Next buttons to move around.
  • Use Submit Exam only when you are ready to finish.

Keyboard controls

  • Press A, B, C, or D to answer.
  • Press Left Arrow for previous question.
  • Press Right Arrow for next question.
  • Press 1-9 to jump within the visible palette groups.

Scoring and feedback

  • You will see total score and subject-by-subject performance.
  • Weak topics are highlighted so you know what to revise.
  • Each exam session creates a fresh randomized paper.
  • Your latest session is stored only in this browser.
